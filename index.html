Foot Placement System

This system guides the player to walk through some rough terrains while having proper foot placement. Core features are as follows:

  • Properly set foot IKWeight to avoid unnatural foot movement
  • Adjust positions of colliders to avoid deformation of bones

Inventory System

I am a big fan of The Elder Scroll V: Skyrim, so I implemented a similar inventory system. My framework ensures the implementation of various Skyrim inventory features, including:

  • Equip and unequip items
  • Pick up and drop items
  • Loot and trade items
  • Store items in containers
  • Auto unequip sold, looted and dropped items
  • View item models in inventory

Unity Extension - Item and Inventory Editor

I developed this editor extension to support data serialization, and easier data manipulation. What I have in mind is to create a tool to modify items and inventories without coding and further editing.

Key features of this tool are as follows:

  • Serialization of items and inventories data to support save and load games
  • Item and inventory data can be modified in both editor mode and play mode
  • Delete all related data when an item is deleted from the database to avoid error
  • Auto process item models and attach them to inventory camera, then show them properly in inventory Camera
  • Support inventory data for prefab object
  • Auto create, modify prefab files and organize them for the project
  • Auto link prefab files: rename and relocation of prefab files will be relinked to the database
  • Auto attach core scripts to items according to item type so they can be properly used

Reactive Combat System - Hit, Dodge, Block and Spell

Reactive combat system will analyze triggered events and use calculated parameters to blend animations. Such mechanism is used to simulate more realistic hit and block reaction.

Target Locking System

When an agent has locked on a target, this agent will use a new coordinate system to calculate movement parameters to blend animations. AI will circle around the player through randomly generated coordinates, using the same system.

Character Control System

The animation package I was using came without a controller (Longsword Animset Pro). So I designed and implemented my own character control system in Unity animator. The camera logic was added after the control system is functional. My system covers basic features of an ARPG:

  • Camera following
  • Rotate, zoom in and zoom out camera
  • Indoor collision check for camera
  • Use animation blending and root motion for character movement
  • Controller for all humanoid agents
